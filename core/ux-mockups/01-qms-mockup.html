DEV-20260508-001 CRITICAL OPEN MATERIAL

Moisture content OOS — Paracetamol API lot INV-2026-0892

Deviation Details

Source

QC Sampling — SR-20260508-017

Detected By / At

qc.analyst · 2026-05-08 09:42

Department

Quality Control

Material / Lot

Paracetamol API (MAT-0023) · INV-2026-0892

Description

During incoming QC sampling (SR-20260508-017), moisture content of Paracetamol API (Lot INV-2026-0892, Vendor Batch VB-2026-8831) tested at 6.8% w/w against a specification limit of 2.0–4.0% w/w. Result exceeds upper limit by 2.8 percentage points. Karl Fischer titration performed in triplicate; all three readings consistent (6.7%, 6.8%, 6.9%). Instrument calibration verified valid.

Immediate Action Taken

Lot INV-2026-0892 placed under QUARANTINE hold. Material physically segregated and labelled "DO NOT USE – OOS Investigation." Warehouse Supervisor notified. Supplier (Cipla Ltd, Vendor BU: Cipla-API-HYD) notified via email at 10:15.

Root Cause (under investigation)

Root cause not yet established — investigation in progress.
